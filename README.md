# Chatbot

Welcome to the Chatbot project! This is a simple chatbot built using JavaScript and HTML.

## Table of Contents

- [Introduction](#introduction)
- [Features](#features)
- [Getting Started](#getting-started)
- [Usage](#usage)

## Introduction

This Chatbot project is designed to provide a basic interactive chat experience. Users can communicate with the chatbot, and it will generate responses using the OpenAI GPT-3.5-turbo model.

## Features

- Interactive chat interface
- Integration with the OpenAI GPT-3.5-turbo model
- Simple UI with toggler and chatbox

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/asish-sutar/chatbot.git
   ```

2. Open the `index.html` file in your web browser.

3. Provide your OpenAI API key in the `script.js` file:

   ```javascript
   const API_KEY = "YOUR_OPENAI_API_KEY";
   ```

4. Start chatting with the bot!

## Usage

- Click the chatbot toggler to open/close the chatbox.
- Enter a message in the chat input and press Enter or click the send button to send the message.
- The chatbot will respond with a generated message using the OpenAI GPT-3.5-turbo model.
